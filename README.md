# RRPrettier

A tool to prettify data of python objects

A simple example:

```python
import rrprettier
data={
"name": "John Doe",
"age": 50,
"city": "New York",
"isStudent": True,
"courses": ["Math", "Science", "English"],
"address": {
    "street": "123 Main St",
    "zipCode": "10001"
}
}
rrprettiernew.pprint(data)
```
